## Proyecto MarvelComic

Aplicación Realizada con el framework Ionic, fue creada para visualizar los comics de Marvel mediante su Api gratuita (http://developer.marvel.com/).

Realizado con:
```bash
•ionic 3.7.0
•node 6.11.2
•cordova 7.0.1 
```
